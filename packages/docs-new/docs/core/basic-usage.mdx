# Basic Usage

Learn how to integrate Google reCAPTCHA using the core package utilities.

## Utility Functions

The core package provides essential utility functions for reCAPTCHA integration:

### generateGoogleReCaptchaScriptSrc

Generates the script source URL for loading Google reCAPTCHA:

```typescript
import { generateGoogleReCaptchaScriptSrc } from '@google-recaptcha/core';

const scriptSrc = generateGoogleReCaptchaScriptSrc({
  isEnterprise: false,
  render: 'your-site-key',
  hl: 'en',
  host: 'google.com'
});

// Load the script
const script = document.createElement('script');
script.src = scriptSrc;
script.async = true;
script.defer = true;
document.head.appendChild(script);
```

#### Parameters

| Parameter      | Type                                        | Required | Description                         |
| -------------- | ------------------------------------------- | -------- | ----------------------------------- |
| `isEnterprise` | `boolean`                                   | Yes      | Whether to use reCAPTCHA Enterprise |
| `render`       | `string`                                    | Yes      | Your reCAPTCHA site key             |
| `hl`           | `string`                                    | No       | Language code (e.g., 'en', 'ru')    |
| `host`         | `'google.com' \| 'recaptcha.net'`           | No       | Host to load from                   |
| `badge`        | `'inline' \| 'bottomleft' \| 'bottomright'` | No       | Badge position                      |

### generateGoogleReCaptchaHiddenBadgeStyles

Generates CSS styles to hide the reCAPTCHA badge:

```typescript
import { generateGoogleReCaptchaHiddenBadgeStyles } from '@google-recaptcha/core';

const hiddenStyles = generateGoogleReCaptchaHiddenBadgeStyles();

// Apply styles
const style = document.createElement('style');
style.textContent = hiddenStyles;
document.head.appendChild(style);
```

<Callout type='warning'>
  When hiding the reCAPTCHA badge, you must include the reCAPTCHA terms in your privacy policy as
  required by Google's terms of service.
</Callout>

## Complete Examples

### V3 Implementation

```typescript
import { generateGoogleReCaptchaScriptSrc } from '@google-recaptcha/core';

// 1. Generate and load script
const scriptSrc = generateGoogleReCaptchaScriptSrc({
  isEnterprise: false,
  render: 'your-v3-site-key'
});

const script = document.createElement('script');
script.src = scriptSrc;
script.async = true;
script.defer = true;

script.onload = () => {
  // 2. Execute reCAPTCHA when script loads
  grecaptcha.ready(() => {
    grecaptcha.execute('your-v3-site-key', { action: 'submit' }).then((token) => {
      console.log('reCAPTCHA token:', token);
      // Send token to your server for verification
    });
  });
};

document.head.appendChild(script);
```

### V2 Invisible Implementation

```typescript
import { generateGoogleReCaptchaScriptSrc } from '@google-recaptcha/core';

// 1. Load script
const scriptSrc = generateGoogleReCaptchaScriptSrc({
  isEnterprise: false,
  render: 'explicit'
});

const script = document.createElement('script');
script.src = scriptSrc;
script.async = true;
script.defer = true;

script.onload = () => {
  grecaptcha.ready(() => {
    // 2. Render invisible reCAPTCHA
    const widgetId = grecaptcha.render('recaptcha-container', {
      sitekey: 'your-v2-invisible-site-key',
      size: 'invisible',
      callback: (token) => {
        console.log('reCAPTCHA token:', token);
        // Send token to your server
      }
    });

    // 3. Execute when needed
    document.getElementById('submit-button').addEventListener('click', () => {
      grecaptcha.execute(widgetId);
    });
  });
};

document.head.appendChild(script);
```

### V2 Checkbox Implementation

```typescript
import { generateGoogleReCaptchaScriptSrc } from '@google-recaptcha/core';

// 1. Load script
const scriptSrc = generateGoogleReCaptchaScriptSrc({
  isEnterprise: false,
  render: 'explicit'
});

const script = document.createElement('script');
script.src = scriptSrc;
script.async = true;
script.defer = true;

script.onload = () => {
  grecaptcha.ready(() => {
    // 2. Render checkbox
    grecaptcha.render('recaptcha-container', {
      sitekey: 'your-v2-checkbox-site-key',
      theme: 'light',
      size: 'normal',
      callback: (token) => {
        console.log('reCAPTCHA token:', token);
        // Enable form submission
      },
      'expired-callback': () => {
        console.log('reCAPTCHA expired');
        // Disable form submission
      }
    });
  });
};

document.head.appendChild(script);
```

## Enterprise Mode

For enterprise implementations, set `isEnterprise: true`:

```typescript
const scriptSrc = generateGoogleReCaptchaScriptSrc({
  isEnterprise: true,
  render: 'your-enterprise-site-key'
});

// Enterprise reCAPTCHA execution with action
grecaptcha.enterprise
  .execute('your-enterprise-site-key', {
    action: 'custom_action'
  })
  .then((token) => {
    // Handle enterprise token
  });
```
