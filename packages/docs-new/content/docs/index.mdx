# 🔑 Google reCAPTCHA

A powerful tool that easily and quickly adds Google reCAPTCHA protection for your website or application, supporting all versions and frameworks.

## 🦉 Philosophy

**🔑 Google reCAPTCHA** is a comprehensive library designed to seamlessly integrate Google reCAPTCHA into your web applications. Our main differentiator is complete support for all current versions of Google reCAPTCHA across multiple frameworks.

## ✨ Key Features

- **🛡️ Complete Version Support** - Full support for v3, v2-checkbox, and v2-invisible
- **🏢 Enterprise Ready** - Built-in support for [Google reCAPTCHA Enterprise](https://cloud.google.com/recaptcha-enterprise/docs/overview)
- **📦 TypeScript First** - Full type safety out of the box
- **🚀 SSR Compatible** - Works seamlessly with server-side rendering
- **🎯 Framework Agnostic** - Available for React, Vue, and vanilla JavaScript

## 🔒 reCAPTCHA Versions

### V3 - Invisible Protection

[Google reCAPTCHA v3](https://developers.google.com/recaptcha/docs/v3) provides frictionless protection by analyzing user behavior and returning a risk score (0.0-1.0) for each request.

### V2 Invisible - Background Analysis

[Google reCAPTCHA v2 invisible](https://developers.google.com/recaptcha/docs/display) runs in the background, analyzing user behavior without requiring puzzle solving.

### V2 Checkbox - Interactive Challenge

[Google reCAPTCHA v2 checkbox](https://developers.google.com/recaptcha/docs/display) provides the familiar "I'm not a robot" checkbox with optional additional challenges.

<Tabs items={['React', 'Vue', 'Core JavaScript']}>
  <Tab value="React">
    ### React Integration

    Perfect for React and Next.js applications with hooks and components.

    **Installation**
    ```bash
    npm install @google-recaptcha/react
    ```

    **Quick Setup**
    ```jsx
    import { GoogleReCaptchaProvider, useGoogleReCaptcha } from '@google-recaptcha/react';

    function App() {
      return (
        <GoogleReCaptchaProvider type="v3" siteKey="your-site-key">
          <ContactForm />
        </GoogleReCaptchaProvider>
      );
    }

    function ContactForm() {
      const { executeRecaptcha } = useGoogleReCaptcha();

      const handleSubmit = async () => {
        if (executeRecaptcha) {
          const token = await executeRecaptcha('contact_form');
          // Send token to your server
        }
      };

      return <button onClick={handleSubmit}>Submit</button>;
    }
    ```

    **Key Features**
    - ⚛️ React hooks for easy integration
    - 🧩 Pre-built components
    - 🔄 Context-based state management
    - 🚀 Next.js SSR support

    ### Core JavaScript

    Framework-agnostic utilities for any JavaScript application.

    **Installation**
    ```bash
    npm install @google-recaptcha/core
    ```

    **Quick Setup**
    ```typescript
    import { generateGoogleReCaptchaScriptSrc } from '@google-recaptcha/core';

    // Load reCAPTCHA script
    const scriptSrc = generateGoogleReCaptchaScriptSrc({
      isEnterprise: false,
      render: 'your-site-key'
    });

    const script = document.createElement('script');
    script.src = scriptSrc;
    script.async = true;
    document.head.appendChild(script);

    // Execute reCAPTCHA
    script.onload = async () => {
      await grecaptcha.ready();
      const token = await grecaptcha.execute('your-site-key', {
        action: 'contact_form'
      });
      // Send token to your server
    };
    ```

## 🚀 Quick Start Guide

1. **Choose Your Framework** - Select React, Vue, or Core JavaScript above
2. **Install Package** - Use npm, yarn, or pnpm to install
3. **Get reCAPTCHA Keys** - Visit [Google reCAPTCHA Admin](https://www.google.com/recaptcha/admin)
4. **Follow Integration** - Use the framework-specific setup above
5. **Implement Server Verification** - Always verify tokens server-side
