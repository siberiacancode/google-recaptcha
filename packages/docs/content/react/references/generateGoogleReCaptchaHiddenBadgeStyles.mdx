---
title: generateGoogleReCaptchaHiddenBadgeStyles
description: Utility function for generating CSS styles to hide the Google reCAPTCHA badge
---

```tsx
const styles = generateGoogleReCaptchaHiddenBadgeStyles();
```

## Return

Returns a string containing CSS styles that hide the Google reCAPTCHA badge.

<Callout>
  This utility is useful when you want to hide the reCAPTCHA badge while still complying with
  Google's terms of service. Make sure to include a visible reCAPTCHA badge or a text notice about
  reCAPTCHA usage on your page.
</Callout>
