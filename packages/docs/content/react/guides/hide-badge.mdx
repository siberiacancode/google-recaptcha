## Implementation

To hide the reCAPTCHA badge, you need to:

1. Add CSS styles to hide the badge
2. Add a visible notice about reCAPTCHA usage
3. Configure the provider with the "hidden" badge option

### Add Styles

Import and use the utility function to generate CSS styles for hiding the badge:

```tsx
import { generateGoogleReCaptchaHiddenBadgeStyles } from '@google-recaptcha/core';

// Add styles to document head
const styleElement = document.createElement('style');
styleElement.textContent = generateGoogleReCaptchaHiddenBadgeStyles();
document.head.appendChild(styleElement);
```

### Add Notice

Add a visible notice about reCAPTCHA usage to comply with Google's terms of service:

```tsx
function ReCaptchaNotice() {
  return (
    <div className='recaptcha-notice'>
      This site is protected by reCAPTCHA and the Google{' '}
      <a href='https://policies.google.com/privacy'>Privacy Policy</a> and{' '}
      <a href='https://policies.google.com/terms'>Terms of Service</a> apply.
    </div>
  );
}
```

### Configure Provider

Set the badge position to "hidden" in your provider configuration:

```tsx
<GoogleReCaptchaProvider
  type='v3'
  siteKey='your_site_key'
  explicit={{
    badge: 'hidden'
  }}
>
  {children}
</GoogleReCaptchaProvider>
```

## Important Notes

1. **Terms of Service Compliance**: When hiding the reCAPTCHA badge, you must include a visible notice about reCAPTCHA usage on your page. This is required by Google's Terms of Service.

2. **Accessibility**: Ensure your reCAPTCHA notice is accessible and visible to all users. Consider adding it to your site's footer or near forms that use reCAPTCHA.

3. **Mobile Considerations**: The notice should be clearly visible on both desktop and mobile devices.

## Complete Example

Here's a complete example of implementing hidden badge with proper notice:

```tsx
import { GoogleReCaptchaProvider } from '@google-recaptcha/react';
import { generateGoogleReCaptchaHiddenBadgeStyles } from '@google-recaptcha/core';

// Add styles
const styleElement = document.createElement('style');
styleElement.textContent = generateGoogleReCaptchaHiddenBadgeStyles();
document.head.appendChild(styleElement);

function App() {
  return (
    <GoogleReCaptchaProvider
      type='v3'
      siteKey='your_site_key'
      explicit={{
        badge: 'hidden'
      }}
    >
      <div className='app'>
        {/* Your app content */}
        <footer className='footer'>
          <div className='recaptcha-notice'>
            This site is protected by reCAPTCHA and the Google{' '}
            <a href='https://policies.google.com/privacy'>Privacy Policy</a> and{' '}
            <a href='https://policies.google.com/terms'>Terms of Service</a> apply.
          </div>
        </footer>
      </div>
    </GoogleReCaptchaProvider>
  );
}
```
